# 🚀 Smart Task Board

A modern, containerized 3-tier web application built to manage tasks efficiently. 

## 🏗️ Architecture
This project follows a professional 3-tier architecture:
* **Frontend (User Interface):** Next.js with Tailwind CSS
* **Backend (API):** Python with FastAPI
* **Database (Storage):** PostgreSQL
* **Infrastructure:** Docker & AWS EC2

## 📂 Project Structure
* `/frontend` - Contains the Next.js UI code.
* `/backend` - Contains the FastAPI Python code.
* `/database` - Contains database initialization scripts.
